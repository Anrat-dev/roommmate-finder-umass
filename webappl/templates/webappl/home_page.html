Overview of the project

The intent of this project was to develop a roommate matching application for students, alumni, and faculty of UMass Amherst. The application allows UMass students to find compatible roommates and get their contact information. There currently exists a roommate matching application for UMass, however our application aims to make improvements over this application. We hope to improve the quality of roommate match results and also improve the UI.
